# Architecture Decision Records (ADR)

Décisions d'architecture significatives, une par fichier, numérotées et
immuables (un changement = un nouvel ADR qui supersède le précédent).

## Index

| #    | Titre                                                            | Statut   |
| ---- | --------------------------------------------------------------- | -------- |
| 0001 | [SVG + Theatre.js plutôt que Rive](0001-svg-theatrejs-vs-rive.md) | Accepté  |

## Format

Chaque ADR suit la trame : **Contexte → Décision → Justification →
Conséquences → Alternatives**.
